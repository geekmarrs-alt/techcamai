How it works

Up in three steps

From bare Pi to monitoring in under ten minutes. Tested on Raspberry Pi OS Lite.

1
Install on your Pi

Run the one-liner install script. Docker is installed automatically if missing. The compose stack starts three services: API, worker, and Watchtower.

2
Add your cameras

Open the operator console at :8000. Scan your LAN, pick cameras, enter credentials, and test a live snapshot — all in the browser.

3
Get alerts with clips

The background worker polls each camera, runs detection, fires alerts with thumbnails to your inbox, and captures a short MP4 clip for playback.

Raspberry Pi OS Lite or Desktop · Ethernet recommended for first run

Install one-liner
# Replace HOST:PORT with your dev machine or file server curl -fsSL http://HOST:PORT/install.sh | sudo bash -s -- \ --src http://HOST:PORT/techcamai.tgz \ --camera-urls \ "http://user:pass@192.168.0.30/ISAPI/Streaming/channels/1/picture" # Then open the console at: http://<pi-ip>:8000/

Docker is installed automatically if missing. Stack runs under /opt/techcamai.
