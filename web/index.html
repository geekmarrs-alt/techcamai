What it does
Everything an operator needs
📷

Live camera wall

See all your cameras at a glance with auto-refreshing snapshots. Status indicators show online/offline instantly.

🔔

Instant AI alerts

Detect people, vehicles, and motion across all cameras. Alerts fire in seconds with snapshot thumbnails attached.

🎬

Clip playback

Every alert triggers automatic clip capture. Review and play back MP4 clips directly in the operator console.

🤖

AI assist rail

Ask questions about your cameras in plain English. Scene summaries and voice control coming in the next tier.

🔍

LAN camera discovery

Scan your network and find cameras automatically. Supports Hikvision ISAPI, ONVIF, RTSP, digest and basic auth.

⚡

Runs on a Pi

Deploy on a Raspberry Pi on your own LAN. No cloud dependency, no subscription required for the community tier.

One command to deploy

Runs on any Linux server or Raspberry Pi. Docker required.

# Clone and start TECHCAMAI
git clone https://github.com/geekmarrs-alt/techcamai.git && cd techcamai
cp .env.example .env
docker compose up -d
